# Simple Webpage UI Design

Project work for Coursera course [HTML and CSS in depth](https://www.coursera.org/learn/html-and-css-in-depth).

Part of the code is generated by AI agents. 


## Live preview
Open `index.html` in a browser or serve the folder locally.

Or preview this page via [Github Pages](https://zhengyeve.github.io/frontend-coursera/).

## Quick start

Clone or copy files into your project directory (this repository root should contain `index.html`, `css/`, and `assets/`). Double-click `index.html` to open in your browser.
